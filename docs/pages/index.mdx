# @textea/json-viewer

### NPM
```shell
npm install @textea/json-viewer
```
### Yarn
```shell
yarn add @textea/json-viewer
```
### PNPM
```shell
pnpm add @textea/json-viewer
```

### Basic Example

```tsx
import { JsonViewer } from '@textea/json-viewer'

const object = { /* my json object */ }
const Component = () => (<JsonViewer value={object}/>)
```
